# UI Sound Effects

This directory contains user interface sound effects.

## Expected Files

### Button Interactions
- `button_hover.ogg` - Button hover sound
- `button_click.ogg` - Button click sound
- `button_disabled.ogg` - Clicking disabled button
- `button_back.ogg` - Back button sound

### Card Interactions
- `card_hover.ogg` - Hovering over card
- `card_select.ogg` - Selecting a card
- `card_place.ogg` - Placing card on battlefield
- `card_invalid.ogg` - Invalid card placement
- `card_draw.ogg` - Drawing a new card
- `card_shuffle.ogg` - Deck shuffling sound

### Menu Sounds
- `menu_open.ogg` - Menu opening
- `menu_close.ogg` - Menu closing
- `tab_switch.ogg` - Switching tabs
- `dropdown_open.ogg` - Dropdown opening
- `dropdown_close.ogg` - Dropdown closing
- `slider_move.ogg` - Moving a slider
- `checkbox_check.ogg` - Checking a checkbox
- `checkbox_uncheck.ogg` - Unchecking a checkbox

### Notifications
- `notification_info.ogg` - Information notification
- `notification_success.ogg` - Success notification
- `notification_warning.ogg` - Warning notification
- `notification_error.ogg` - Error notification
- `achievement_unlock.ogg` - Achievement unlocked

### Game State
- `game_start.ogg` - Game starting
- `game_pause.ogg` - Game paused
- `game_resume.ogg` - Game resumed
- `turn_start.ogg` - Turn starting
- `countdown_tick.ogg` - Countdown timer tick
- `countdown_final.ogg` - Final countdown sound

### Economy
- `elixir_gain.ogg` - Gaining elixir
- `elixir_spend.ogg` - Spending elixir
- `elixir_full.ogg` - Elixir bar full
- `gold_gain.ogg` - Gaining gold
- `gold_spend.ogg` - Spending gold
- `purchase_success.ogg` - Successful purchase
- `purchase_fail.ogg` - Failed purchase

### Deck Building
- `card_add_to_deck.ogg` - Adding card to deck
- `card_remove_from_deck.ogg` - Removing card from deck
- `deck_save.ogg` - Saving deck
- `deck_load.ogg` - Loading deck
- `deck_invalid.ogg` - Invalid deck configuration

### Battle UI
- `tower_damage.ogg` - Tower taking damage
- `tower_destroy.ogg` - Tower destroyed
- `castle_damage.ogg` - Castle taking damage
- `overtime_start.ogg` - Overtime starting
- `victory_announcement.ogg` - Victory announcement
- `defeat_announcement.ogg` - Defeat announcement

### Chat/Social
- `message_send.ogg` - Sending message
- `message_receive.ogg` - Receiving message
- `emote_play.ogg` - Playing emote
- `friend_online.ogg` - Friend comes online

## File Format
- Format: OGG Vorbis (.ogg) or MP3 (.mp3)
- Duration: 0.1-1 second for most UI sounds
- Sample rate: 44.1 kHz
- Keep file sizes small for quick loading
