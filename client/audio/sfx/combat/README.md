# Combat Sound Effects

This directory contains combat-related sound effects.

## Expected Files

### Melee Weapons
- `sword_swing.ogg` - Sword swing sound
- `sword_hit.ogg` - Sword hitting target
- `sword_clash.ogg` - Swords clashing together
- `axe_swing.ogg` - Axe swing sound
- `axe_hit.ogg` - Axe hitting target
- `spear_thrust.ogg` - Spear thrust sound
- `spear_hit.ogg` - Spear hitting target

### Ranged Weapons
- `arrow_draw.ogg` - Bow drawing sound
- `arrow_release.ogg` - Arrow being released
- `arrow_fly.ogg` - Arrow flying through air (loop)
- `arrow_hit.ogg` - Arrow hitting target
- `crossbow_reload.ogg` - Crossbow reloading
- `crossbow_fire.ogg` - Crossbow firing

### Magic/Spells
- `fireball_cast.ogg` - Fireball casting
- `fireball_travel.ogg` - Fireball traveling (loop)
- `fireball_impact.ogg` - Fireball explosion
- `lightning_cast.ogg` - Lightning spell cast
- `lightning_strike.ogg` - Lightning strike impact
- `heal_cast.ogg` - Healing spell sound
- `buff_apply.ogg` - Buff application sound
- `debuff_apply.ogg` - Debuff application sound

### Explosions
- `explosion_small.ogg` - Small explosion
- `explosion_medium.ogg` - Medium explosion
- `explosion_large.ogg` - Large explosion
- `catapult_impact.ogg` - Catapult projectile impact

### Defensive Sounds
- `shield_block.ogg` - Shield blocking attack
- `armor_hit.ogg` - Hit on armor
- `dodge_whoosh.ogg` - Dodging sound
- `parry.ogg` - Parrying sound

### Critical/Special
- `critical_hit.ogg` - Critical hit sound
- `miss_whoosh.ogg` - Attack missing

## Variations
Consider having 2-3 variations of frequently used sounds (like sword_hit_1.ogg, sword_hit_2.ogg, sword_hit_3.ogg) to avoid repetition.

## File Format
- Format: OGG Vorbis (.ogg) or MP3 (.mp3)
- Duration: Most combat sounds should be 0.5-2 seconds
- Sample rate: 44.1 kHz or 48 kHz
