## Post-Processing Notes

### Background Removal
1. **remove.bg** - Automatic (https://remove.bg)
2. **Photoshop** - Magic Wand or "Select Subject"
3. **GIMP** - Free alternative
4. **Photopea** - Free online (https://photopea.com)

### Recommended Workflow
1. **Generate** - Use DALL-E prompts above
2. **Download** - Get at 1024x1024 resolution
3. **Remove background** - Ensure transparency
4. **Resize** - Scale to target dimensions listed above
5. **Save as PNG** - With alpha channel
6. **Place in directory** - `/client/assets/sprites/units/`

### File Format Requirements
- **Format**: PNG with transparency (alpha channel)
- **Color Mode**: RGBA (32-bit)
- **Naming**: lowercase, underscores (snake_case)
- **Size**: Match minimum dimensions listed above

4. **Musketeer** - Long range single target

### Lower Priority
7. **Minions** - Flying ranged units
